
# Chapter 1 Notes — Introduction to ML Engineering

## Main idea

Machine learning engineering is about turning machine learning models into reliable software systems.

A notebook model is not enough. A production-style ML system needs:

- Clear business problem
- Reliable data pipeline
- Reproducible training
- Model evaluation
- Model persistence
- Deployment path
- Monitoring and retraining plan

## Data scientist vs ML engineer

A data scientist focuses more on analysis, modelling, and insight.

An ML engineer focuses more on:

- Translating research/model code into robust software
- Designing architecture around the model
- Productionising the model so it creates value
